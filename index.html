Tentang Saya

Halo! Saya seorang pembelajar seumur hidup yang passionate tentang teknologi dan AI. Meskipun berlatar belakang ilmu sosial dan humaniora, saya percaya bahwa siapa pun bisa belajar coding dan menggunakan teknologi untuk mewujudkan ide-ide mereka.

Saat ini saya sedang dalam perjalanan untuk menjadi seorang "vibe coder" - Pelajari lebih lanjut tentang filosofi Vibe Coder - seseorang yang mungkin tidak punya gelar CS, tapi bisa memanfaatkan AI tools seperti Claude dan Gemini serta LLM lainnya untuk membangun aplikasi yang bermanfaat. Saya percaya bahwa coding bukan hanya untuk engineer, tapi untuk siapa saja yang punya ide dan kemauan untuk belajar.

Filosofi saya sederhana: build first, learn later. Langsung praktik, baru kemudian memahami teorinya. Dengan pendekatan ini, belajar coding jadi lebih menyenangkan dan tidak terasa seperti beban!
